== Description ==

flicknpress is a WordPress plugin that lets you attach a cropped photo from Flickr right inside your blog post.

== Installation ==

1. Upload `flicknpress` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Can I search through only the photos on my personal Flickr account? =

The flicknpress plugin does not currently support searching by individual user.
